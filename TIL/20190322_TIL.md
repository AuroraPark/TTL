### 제한된 지네릭클래스
```
class FruitBox<T extends Fruit>
```
> Fruit의 자손만 타입으로 지정가능

> extends를 사용

### 와일드 카드 '?'
```
<? extends T>	T와 그 자손들
<?>				제한없음
```
타입을 여러개 받을 수 있게 됨
```
new FruitBox<Apple>
new FruitBox<Grape>
```

### 지네릭 메서드
지네릭 메서드를 호출할 때, 타입변수에 타입을 대입
