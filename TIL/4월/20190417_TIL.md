오늘의 SQL

### 1. OUTER JOIN

> WHERE e.id **(+)** = c.sales_rep_id  -- LEFT OUTER JOIN

```
select e.name 사원명, e.id, c.name 고객명 
From s_emp e, s_customer c
WHERE e.id (+) = c.sales_rep_id  -- LEFT OUTER JOIN
--WHERE e.id = c.sales_rep_id (+) -- RIGHT OUTER JOIN
ORDER BY 2;
```
